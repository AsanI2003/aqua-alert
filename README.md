💧 Aqua Alert <br>
Stay Hydrated, Stay Healthy. <br>
Aqua Alert is a cross-platform mobile application designed to help users track their daily water intake and achieve their hydration goals through an intuitive interface and smart reminders.


📲 Download & Install (Android) <br>
You can download the latest preview version of the app directly from the link below. <br>
https://expo.dev/accounts/asan2003/projects/Aqua-Alert/builds/00c2509e-4473-4341-b2c7-e8093882d5b0

Note: Since this is a preview build, your phone may show a "blocked by Play Protect" warning. You can safely click "Install Anyway."


✨ Features <br>
Persistent Auth: Secure login that remembers you even after closing the app. <br>
Daily Tracking: Log water intake with custom amounts. <br>
Progress Stats: View your weekly hydration trends. <br>
Smart Layout: Optimized navigation using Expo Router. <br>
System Sync: Automatic Dark/Light mode support via NativeWind.


🛠️ Tech Stack <br>
Framework: Expo (React Native) <br>
Navigation: Expo Router (v3) <br>
Database/Auth: Firebase (Firestore & Auth) <br>
Styling: NativeWind (Tailwind CSS) <br>
Icons: Lucide React Native


🚀 Development Setup

Clone & Install: <br>
git clone https://github.com/your-username/aqua-alert-project.git <br>
cd aqua-alert-project <br>
npm install

Environment Variables: <br>
Create a .env file and add your EXPO_PUBLIC_FIREBASE_ keys.

Run Locally: <br>
npx expo start
